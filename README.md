# 💰 FD Calculator

A simple and user-friendly **Fixed Deposit (FD) Calculator** built using **HTML, CSS, and JavaScript**.
It helps users calculate their **maturity amount, interest earned, and total deposit value** easily.

## 📌 Features

✨ Easy-to-use interface

✨ Supports different compounding frequencies
✨ Responsive design (works on mobile 📱 & desktop 💻)
✨ Keyboard-friendly navigation (Enter key support)

---

## 🧮 How It Works

The calculator uses the compound interest formula:

A = P(1 + r/n)^(nt)

Where:

* **P** = Principal amount
* **r** = Interest rate
* **t** = Time (years)
* **n** = Compounding frequency
  * **Yearly** 	n is 1
  * **Half-Yearly**	n is 2
  * **Quarterly** n is 4
  * **Monthly** n is 12
